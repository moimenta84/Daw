# Práctica de Git

## Descripción
Repositorio creado para la práctica de **Git y control de versiones**.  
El objetivo es aprender y aplicar los comandos básicos y el flujo de trabajo habitual con Git.

---

## Objetivos de la práctica
- Inicializar un repositorio Git.
- Gestionar versiones del proyecto.
- Trabajar con ramas.
- Realizar commits de forma correcta.
- Subir el proyecto a un repositorio remoto.

---

## Tecnologías utilizadas
- Git
- GitHub (repositorio remoto)
- Editor de texto / IDE (VS Code u otro)

---

## Estructura del proyecto
- `README.md` → documentación del proyecto.
- Archivos de práctica según el ejercicio propuesto.

---

## Comandos principales utilizados
- `git init`
- `git status`
- `git add .`
- `git commit -m "mensaje"`
- `git branch`
- `git checkout -b nombre_rama`
- `git merge`
- `git remote add origin`
- `git push -u origin main`

---

## Flujo de trabajo
1. Crear o modificar archivos.
2. Añadir cambios al área de preparación.
3. Confirmar cambios con commit.
4. Subir cambios al repositorio remoto.
